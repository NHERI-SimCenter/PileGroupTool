    Detailed Description

    The ResultPlotSuper class is an abstract class implementing simple line plots.

    This class provides library-independent data exchange and data management for result plots.

    Any library-dependent functionality needs to be implemented by a derived class. The derived class must implement the \method refresh method.

    Note: Implementations exist for both the Qwt and the QCustomPlot libraries.

    See also ResultPlotQwt and ResultPlotQCP.
